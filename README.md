<h2>🛠️ Installation Steps:</h2>

<p>1. Clone this repo</p>

```
git clone git@github.com:bibekdev/netflix-ui-clone-nextjs.git
```

<p>2. Install dependency</p>

```
yarn
```

<p>3. Run dev server</p>

```
yarn dev
```
